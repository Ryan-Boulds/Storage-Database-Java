Aisin Inventory Manager
Requirements:
- Java 8 (Not tested on later versions)
- A valid .accdb database file
- Read/write permissions for the database file and logs/ directory
Instructions:
1. Place the .accdb file in a writable directory.
2. Run: java -jar Storage-Database-Java-1.0-SNAPSHOT.jar
3. Select the .accdb file when prompted.
4. Ensure the logs/ directory is writable for logging.
Report issues to: r-boulds@aisinil.com
